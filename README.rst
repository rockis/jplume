==============================
JPlume Java Lightweight Web Framework
==============================

Features:
--------

 * Tiny 
 * Fast
 * Little dependence
 * No XML (except web.xml)
 * Pure Java
 * REST-style URLs

Requirements:
------------
 * Java7+
 * Maven2


Dependences:
------------
 * slf4j
 * servlet api 2.x
 * freemarker 2.3.x
 * json-lib
